# Cyber Rogue Wars

A turn-based roguelite strategy game inspired by Cyber Wars.

## Factions
- **Voidlings** — Lovecraftian eldritch horror
- **The Confederation** — Brotherhood of Steel style militarists  
- **Conglomerate of Orion (COO)** — Merchant hyperproduction AIs

## Turn Structure
1. Planning Phase — all players choose an action card
2. Resolution Phase — cards resolve in turn order
3. Unit Phase — move/attack with units on the grid
4. Turn order rotates (last becomes first)

## Roguelite Layer
After each level (beating an NPC), draft 1 of 4 random upgrades.
